## Uso
`python cli.py -G <generador> -S <seed> <...args> -N <int> -T <test>`

-N: cantidad de números que vamos a generar

-S: Semilla del generador

-T: Test a realizar
    bitmap

-G: Generador utilizado

    cm: Cuadrados Medios
        -n: precisión 
    glc: GLC
        -m: modulo
        -a: multiplicador
        -c: incremento
